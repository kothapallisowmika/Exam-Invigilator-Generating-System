# Exam-Invigilator-Generating-System
The Exam Invigilator Allocation System is a web-based application designed to automate the process of assigning invigilators for examinations. It reduces manual effort by allocating faculty members based on their availability while preventing scheduling conflicts.
**Features**
Automatic invigilator allocation
Faculty availability management
Conflict-free scheduling
Admin dashboard for managing faculty and exam schedules
Secure database integration
Simple and user-friendly interface
**Technologies Used**
Python
Flask
MySQL
HTML
CSS
JavaScript
**How It Works**
The admin adds faculty details.
Faculty members update their available time slots.
The admin schedules examinations by selecting the date, time, and hall.
The system displays available invigilators for the selected schedule.
The admin assigns an invigilator, and the system updates the database automatically.
